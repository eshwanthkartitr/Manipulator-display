Objectives for the Project

  • Define the kinematics of the hybrid 5 DOF manipulator, outlining its structure and degrees of freedom.
  • Formulate transformation matrix equations to establish the relationship between joint angles and end-effector position and orientation (forward kinematics)
  • Simulate the robot motion for various inputs of the joint angular value using in web using three.js
  • Develop a neural network model for inverse kinematics, with the input being end-effector position and orientation, and the output being joint angles and transformation values.
